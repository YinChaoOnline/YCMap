##已实现的功能
1. 文件菜单，新建、打开、保存、另存为地图文档、添加数据、退出功能
2. 书签加载、书签添加功能、书签管理功能
3. mapcontrol与pagelayoutcontrol控件同步
4. 状态栏信息添加
5. 鹰眼实现
6. 右键菜单实现
